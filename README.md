# McDonald's Sales Analysis Dashboard (Excel)

## Project Overview

This project presents a sales performance dashboard created in Microsoft Excel to analyze key business metrics including revenue, profit, customer count, satisfaction levels, and country-level sales distribution.

The dashboard focuses on KPI tracking and visual storytelling using Excel’s advanced visualization features.

---

## Project Objective

- Monitor overall sales and profit performance  
- Analyze year-over-year sales trends (2021–2022)  
- Evaluate customer satisfaction dimensions  
- Compare country-level sales distribution  
- Present executive-level KPIs in a structured format  

---

## Tools Used

- Microsoft Excel  
  - Pivot Tables  
  - Charts and KPI Cards  
  - Conditional Formatting  
  - Radar Chart  
  - Map Visualization  

---

## Dashboard Components

### Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Number of Customers
- Percentage Performance Indicators

### Trend Analysis
- 2021 vs 2022 monthly sales comparison
- Year-over-year growth visualization

### Customer Satisfaction Analysis
- Speed
- Quality
- Hygiene
- Service
- Availability

### Geographic Analysis
- Country-level sales visualization using map chart

---

## Key Insights

- 2022 shows stronger monthly sales performance compared to 2021.  
- Availability and hygiene score highest in customer satisfaction metrics.  
- Sales distribution varies significantly across countries.  
- KPI tracking allows quick executive-level performance evaluation.
